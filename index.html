Часто задаваемые вопросы

Оптимальный возраст для начала подготовки к школе - 5-6 лет. Для помощи по школьной программе - с 7 лет (1 класс). В каждом случае программа подбирается индивидуально.
Рекомендую 2-3 занятия в неделю по 30-45 минут для дошкольников и 45-60 минут для школьников. График подбирается индивидуально под ваши пожелания и возможности ребёнка.
Для детей 5-7 лет желательно присутствие родителя хотя бы на первых занятиях. Школьники обычно занимаются самостоятельно. После каждого занятия я даю обратную связь родителям.
Для онлайн-занятий нужен только компьютер/планшет с выходом в интернет и установленным Zoom. Все учебные материалы я предоставляю в электронном виде. Для некоторых заданий может понадобиться распечатать листы.
